# TDD

- TDD란 단 테스트가 주도하는 개발을 뜻함
- TDD 절차
  1. 기능단위의 테스트 코드 작성
  2. 테스트가 통과하는 프로덕션 코드 작성
  3. 테스트가 통과하면 프로덕션 코드를 리팩토링



# Junit Test

- 테스트 코드 작성을 도와주는 `java` 의 테스트 프레임워크
- 개발 환경에 Junit 추가 후`@Test` 어노테이션을 사용하여 Junit테스트 가능
