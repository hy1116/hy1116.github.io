## Java Generics

- 타입을 클래스 내부에서 지정하는 것이 아닌 외부에서 사용자에 의해 지정되는 것을 의미

### Generic 장점

- 잘못된 타입이 들어오는걸 컴파일 단계에서 방지
- 클래스 외부에서 타입을 지정할 수 있음

| Type  | 설명    |
| ----- | ------- |
| < T > | Type    |
| < E > | Element |
| < K > | Key     |
| < V > | Value   |
| < N > | Number  |
